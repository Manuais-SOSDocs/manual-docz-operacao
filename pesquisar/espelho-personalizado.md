# Espelho personalizado

A funcionalidade de **Espelhos Personalizados** permite ao usuário gerar impressões de espelhos de caixas de forma rápida e organizada, incluindo a opção de impressão em lote para até 50 caixas. Essa ferramenta é ideal para otimizar a logística de identificação física dos documentos.

#### Acessando a tela

1.  No módulo **Pesquisar**, localize o ícone de **Espelhos Personalizados**. Ele está posicionado acima do bloco que apresenta as pastas do repositório do projeto.<br>

    <figure><img src="../.gitbook/assets/image (84).png" alt=""><figcaption></figcaption></figure>
2. Clique no ícone para abrir a tela de espelhos personalizados.

<figure><img src="../.gitbook/assets/image (83).png" alt=""><figcaption></figcaption></figure>

#### Inserindo os dados das caixas

Na tela, você encontrará dois campos principais:

* **Caixa**: informe o **ID SOS** da caixa.
* **Ano Caixa**: informe o **ano da caixa** correspondente.

Após preencher os campos, clique no botão <img src="../.gitbook/assets/image (82).png" alt="" data-size="original"> para adicionar a caixa à lista.

> ⚠️ Observação: É possível repetir este processo até **50 caixas**, gerando assim uma lista para impressão em lote.

{% hint style="info" %}
#### **Dica de agilidade**

Se você possui uma planilha com vários **ID SOS**, clique no ícone <img src="../.gitbook/assets/image (81).png" alt="" data-size="line">. Isso permite importar os dados diretamente da planilha (Excel, Word ou texto), criando automaticamente a lista do lote, sem a necessidade de inserir cada caixa manualmente.
{% endhint %}

#### Impressão em Lote

Após inserir os dados de todas as caixas desejadas:

1. Clique no botão **Imprimir em Lote**.
2. O sistema gerará automaticamente um **PDF** contendo todos os espelhos, pronto para download no seu computador.

<figure><img src="../.gitbook/assets/image (432).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**Observações importantes**

* **Limite de impressão em lote**: O sistema permite incluir até **50 caixas** por operação. Caso tente adicionar mais, uma mensagem de alerta será exibida para orientar o uso adequado da funcionalidade.
* **Novo layout do espelho**: O layout foi ajustado para garantir que todas as informações da caixa sejam exibidas corretamente em **uma única página**.
{% endhint %}
