# Assinatura em lote nos resultados de pesquisa

{% stepper %}
{% step %}
Na tela de pesquisar, realize a pesquisa e selecione o(s) documento(s) desejado(s).

<figure><img src="../../../../.gitbook/assets/image (192).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Clique no **ícone de checkbox** ao lado do documento. É possível selecionar mais de um documento.

<figure><img src="../../../../.gitbook/assets/image (193).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Após selecionar, clique no **ícone de assinatura** <img src="../../../../.gitbook/assets/image (198).png" alt="" data-size="line"> localizado acima da grade de resultados.

<figure><img src="../../../../.gitbook/assets/image (194).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Uma pop-up será exibida solicitando:

* **Senha de login**;
* (Opcional) Aplicar marca d’água.\
  Após definir os parâmetros, clique em **Assinar**.

<figure><img src="../../../../.gitbook/assets/image (188).png" alt="" width="458"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Caso prefira aplicar **chancela** em vez de assinatura:

Clique no **ícone de chancela** <img src="../../../../.gitbook/assets/image (199).png" alt="" data-size="line"> acima da grade de resultados.

<figure><img src="../../../../.gitbook/assets/image (197).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Confirme a ciência da ação na pop-up exibida clicando em **OK**.

<figure><img src="../../../../.gitbook/assets/image (189).png" alt=""><figcaption></figcaption></figure>


{% endstep %}

{% step %}
Assim que os documentos forem assinados e/ou chancelados, o usuário receberá uma **notificação no e-mail cadastrado**.

<figure><img src="../../../../.gitbook/assets/image (195).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../.gitbook/assets/image (196).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Ação concluída com sucesso!
{% endhint %}
