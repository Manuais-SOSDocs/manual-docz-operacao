# Avaliação

A tela de **Avaliação** é utilizada para a validação dos objetos carregados no sistema.

* **Objetivo:** garantir a conferência e validação do material inserido (imagens e informações associadas), antes que avance para as próximas etapas do fluxo documental.
* **Campo de Etapa Documental:** ao aprovar ou reprovar os objetos, este campo é atualizado automaticamente, funcionando como **gatilho de integração** com outros sistemas e também como mecanismo de **controle de qualidade**.
* **Configuração:** o fluxo de validação e os critérios aplicados podem ser ajustados conforme as necessidades específicas de cada cliente.

#### Como funciona a Tela de Avaliação

Ao acessar a tela de **Avaliação**, o usuário deve seguir os passos abaixo:

1. Utilize o campo _Status do Lote_ para selecionar a situação que deseja consultar (ex.: _Aguardando Aprovação_).
2. No campo _Limite de Registros_, o usuário poderá escolher quantos objetos deseja visualizar por página (ex.: 10, 20, 50, 100).
3. Clique em <mark style="color:blue;">**Pesquisar**</mark> para carregar os resultados de acordo com os filtros aplicados.
