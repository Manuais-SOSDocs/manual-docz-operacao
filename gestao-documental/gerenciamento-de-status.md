# Gerenciamento de Status

Esta tela permite ao usuário realizar a atualização do **status das caixas** ou documentos no processo de gestão documental.

<figure><img src="../.gitbook/assets/image (337).png" alt=""><figcaption></figcaption></figure>



{% stepper %}
{% step %}
Selecione o Status desejado no campo "Status"
{% endstep %}

{% step %}
Insira o número da caixa:

Digite manualmente no campo indicado ou faça a bipagem com o leitor de código de barras.
{% endstep %}

{% step %}
A caixa será adicionada automaticamente à lista de movimentação.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Acompanhe o campo **“Última numeração bipada”**, que mostra o último código lido, ajudando na conferência.
{% endhint %}
