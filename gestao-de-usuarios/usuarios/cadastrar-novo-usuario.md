# Cadastrar Novo Usuário

{% stepper %}
{% step %}
No canto inferior direito da grade de usuários, clique no botão ![](<../../.gitbook/assets/image (217).png>) para abrir o formulário de cadastro.
{% endstep %}

{% step %}
Na **aba Dados**, preencha as informações básicas do usuário, como **nome**, **e-mail** e demais **campos obrigatórios**.

{% hint style="info" %}
Agora o **CPF não é mais obrigatório**.\
Caso ele não seja informado, o e-mail será usado para identificar o usuário e garantir que não haja cadastros repetidos.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (7) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Vá para a aba **Permissões** e selecione as funcionalidades que o usuário poderá acessar, conforme a parametrização do cliente.

<figure><img src="../../.gitbook/assets/image (1) (1) (1) (1).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Saiba o que cada permissão permite fazer — [acesse aqui](permissoes-do-usuario-no-sistema-docz.md).
{% endhint %}
{% endstep %}

{% step %}
Depois de ajustar as **permissões**, vá até a **aba Grupos** para **visualizar os grupos** vinculados ao usuário.

{% hint style="info" %}
**Importante:** apenas usuários com a permissão específica podem **remover um usuário de um grupo**.\
Sem essa permissão, será possível **apenas visualizar** as informações.
{% endhint %}

<figure><img src="../../.gitbook/assets/image (2) (1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Clique em **“Salvar”**.

{% hint style="success" %}
O sistema enviará automaticamente um e-mail ao usuário com o login e uma senha provisória.
{% endhint %}
{% endstep %}

{% step %}
Após salvar, clique no ícone ![](<../../.gitbook/assets/image (216).png>) ao lado do nome do usuário para reabrir o formulário.
{% endstep %}

{% step %}
Na aba **Projetos**, selecione os projetos desejados, clique em **“+” para adicionar** e depois em **Salvar**.

<figure><img src="../../.gitbook/assets/image (3) (1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
No menu lateral, vá em [<kbd>**Configurações > Grupos**</kbd>](../grupos.md).
{% endstep %}

{% step %}
Na aba **Grupos**, selecione o grupo "Clientes" na grid da esquerda.

<figure><img src="../../.gitbook/assets/image (4) (1) (1).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
No campo “Selecione os Usuários”, escolha o usuário recém-criado, clique no nome e depois no botão **“+”** para adicioná-lo ao grupo.

<figure><img src="../../.gitbook/assets/image (407).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Clique em **Salvar**.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
**Pronto!** O usuário está habilitado para acessar o sistema conforme as permissões e projetos definidos.
{% endhint %}
