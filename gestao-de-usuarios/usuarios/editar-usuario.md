# Editar usuário

{% stepper %}
{% step %}
Clique no ícone ![](<../../.gitbook/assets/image (216).png>) ao lado do nome do usuário para reabrir o formulário.
{% endstep %}

{% step %}
Na aba **Dados**, ajuste nome, e-mail ou clique em **Gerar nova senha** para enviar uma senha provisória. Depois, clique em **Salvar**.

<figure><img src="../../.gitbook/assets/image (473).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Na aba **Permissões**, selecione os privilégios desejados. Você também pode inativar o usuário ou gerar uma nova senha provisória. Por fim, clique em **Salvar**.

<figure><img src="../../.gitbook/assets/image (474).png" alt=""><figcaption></figcaption></figure>

<p align="center"><strong>--------------------------------— AÇÕES DISPONIVEIS —---------------------------------</strong></p>

<figure><img src="../../.gitbook/assets/image (362).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Depois na **aba Grupos** para **visualizar os grupos** vinculados ao usuário.

<div align="left"><figure><img src="../../.gitbook/assets/image (475).png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Na aba **Projetos**, selecione os projetos para esse usuário e clique em **Salvar**. Em seguida, peça ao usuário que faça **logout e login** para ativar as mudanças.

<div align="left"><figure><img src="../../.gitbook/assets/image (3) (1) (1) (1).png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}

{% step %}
Habilite a **integração SmartDocs ↔ DocZ** para que o usuário participe do fluxo entre as duas plataformas (SmartDocs e DocZ).

* Marque as opções de integração desejadas
* **Salvar** para concluir

<div align="left"><figure><img src="../../.gitbook/assets/image (476).png" alt="" width="563"><figcaption></figcaption></figure></div>
{% endstep %}
{% endstepper %}

{% hint style="success" %}
**Pronto!** O usuário está atualizado para acessar o sistema conforme as especificações definidos.
{% endhint %}
