# Solicitar O.S. Avulsa

{% hint style="danger" %}
**Atenção:** Esta funcionalidade é **restrita** e requer **permissão específica** para acesso.
{% endhint %}

Os usuários podem **gerar uma Ordem de Serviço fora do fluxo padrão**, utilizada quando **um objeto (caixa ou documento) não é localizado via pesquisa no sistema**.

<figure><img src="../.gitbook/assets/image (135).png" alt=""><figcaption></figcaption></figure>

**Como utilizar:**

{% stepper %}
{% step %}
No menu lateral, acesse **Solicitações > Solicitar O.S. Avulsa**;
{% endstep %}

{% step %}
Preencha o **formulário** com as informações obrigatórias;
{% endstep %}

{% step %}
Após preencher todos os campos, clique em **Enviar Solicitação**.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
A solicitação será processada para avaliação e resposta pela equipe responsável.
{% endhint %}
