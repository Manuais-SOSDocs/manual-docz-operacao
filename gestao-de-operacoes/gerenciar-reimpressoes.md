# Gerenciar Reimpressões

Esta tela é utilizada para aprovar ou recusar solicitações de **reimpressão de etiquetas** feitas no aplicativo.

Sempre que o usuário solicita uma reimpressão no app, o pedido será exibido automaticamente nesta tela e deverá ser aprovado ou recusado por um responsável no DocZ.

<figure><img src="../.gitbook/assets/image (309).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Caso seja necessário gerar um relatório com os dados das solicitações, clique no botão  ![](<../.gitbook/assets/image (310).png>)

O download será iniciado automaticamente no seu computador.
{% endhint %}
