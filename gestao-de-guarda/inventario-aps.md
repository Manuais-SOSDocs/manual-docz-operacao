# Inventário APS

Permitir que o operador realize o inventário físico e virtual dos objetos, associando ou corrigindo etiquetas de endereçamento e controle (SOS).

<figure><img src="../.gitbook/assets/image (267).png" alt=""><figcaption></figcaption></figure>

{% stepper %}
{% step %}
digitada ou bipada (com leitor) a etiqueta de localização física no arquivo (ex: prateleira, caixa, pasta).
{% endstep %}

{% step %}
digitada ou bipada (com leitor) a etiqueta do Identificador SOS
{% endstep %}

{% step %}
Caso a etiqueta do item tenha sido atualizada, informe aqui a versão anterior.

{% hint style="info" %}
### **Última etiqueta BIPADA:**

Campo de exibição automática. Mostra qual foi a última etiqueta lida ou digitada, para facilitar controle em processos contínuos de leitura.
{% endhint %}
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Ação concluída!
{% endhint %}
