# Gerenciamento do Armazém

Esta tela permite cadastrar, visualizar e gerenciar os armazéns do Grupo SOSDocs.

<figure><img src="../.gitbook/assets/image (205).png" alt=""><figcaption></figcaption></figure>

Na parte superior da tela, encontra-se a grid **"Lista de Armazém"**, onde são exibidos os armazéns cadastrados, e suas informações.

#### Ações disponíveis:

<table><thead><tr><th width="125.3636474609375" align="center">Ícone</th><th width="127.1817626953125" align="center">Ação</th><th width="449.0908203125">Descrição</th></tr></thead><tbody><tr><td align="center"><div><figure><img src="../.gitbook/assets/image (255).png" alt=""><figcaption></figcaption></figure></div></td><td align="center">Editar</td><td>Permite atualizar os dados do armazém.</td></tr><tr><td align="center"><div><figure><img src="../.gitbook/assets/image (254).png" alt=""><figcaption></figcaption></figure></div></td><td align="center">Excluir</td><td>Remove o armazém do sistema (após confirmação).</td></tr><tr><td align="center"><div><figure><img src="../.gitbook/assets/image (256).png" alt=""><figcaption></figcaption></figure></div></td><td align="center">Detalhes</td><td> Abre os dados detalhados do armazém.</td></tr><tr><td align="center"><div><figure><img src="../.gitbook/assets/image (257).png" alt=""><figcaption></figcaption></figure></div></td><td align="center">Buscar</td><td>Pesquisar por armazéns usando o campo de busca abaixo da grid.</td></tr></tbody></table>



**Para cadastrar um Armazém**

{% stepper %}
{% step %}
Clique no botão  ![](<../.gitbook/assets/image (206).png>)

_Uma pop-up será aberta no lado esquerdo da tela._
{% endstep %}

{% step %}
Preencha o formulário com as informações sobre a identificação e depois clique em <kbd><mark style="color:blue;">Próximo<mark style="color:blue;"></kbd>

<figure><img src="../.gitbook/assets/image (207).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Notificações de abertura, cancelamento e finalização de OS são enviadas aos e-mails cadastrados no gerenciamento de armazém.
{% endhint %}
{% endstep %}

{% step %}
Preencha o formulário com as informações sobre a localização e depois clique em <kbd><mark style="color:blue;">Próximo<mark style="color:blue;"></kbd>

<figure><img src="../.gitbook/assets/image (208).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Preencha o formulário com as informações sobre a estrutura e depois clique em <kbd><mark style="color:green;">Salvar<mark style="color:green;"></kbd>

<figure><img src="../.gitbook/assets/image (209).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

{% hint style="success" %}
**Ação concluída!**
{% endhint %}
