# Gerenciamento Identificador Cliente

Essa funcionalidade permite **associar a etiqueta original do cliente** (que já vem na caixa) com a **nova etiqueta gerada pela SOS Docs**, garantindo que a busca pela caixa funcione tanto pelos dados antigos (cliente) quanto pelos novos metadados do sistema.

<figure><img src="../.gitbook/assets/image (261).png" alt=""><figcaption></figcaption></figure>

### **Como realizar a associação:**

{% stepper %}
{% step %}
Digite o número de identificação da caixa do cliente.
{% endstep %}

{% step %}
Em seguida, faça a bipagem da nova etiqueta da caixa SOS
{% endstep %}
{% endstepper %}

{% hint style="success" %}
O campo  <kbd>Último número bipado</kbd>  exibirá o código da última caixa SOS lida, confirmando a ação.
{% endhint %}

### **Informações adicionais:**

* A **grid "Caixas Atualizadas"** exibirá um relatório com os identificadores bipados.
* Esse procedimento também pode ser realizado diretamente pelo **aplicativo**.
* Após a associação, será possível **localizar a caixa** tanto pelo identificador original do cliente quanto pelo **ID SOS**.
