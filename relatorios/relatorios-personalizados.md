# Relatórios Personalizados

O módulo de **Relatórios Personalizados** do sistema DocZ permite extrair dados em formato `.xlsx` , com base em filtros específicos como tipo de relatório e período.



{% stepper %}
{% step %}
Antes de tudo, selecione o projeto. Esse passo é fundamental — o campo _Tipo de Relatório_ só será habilitado após essa seleção.

> _Para isso, acesse e configure as informações na aba_ [_**`PROJETOS`**_](../projetos/)
{% endstep %}

{% step %}
Clique no campo “Tipo Relatório” e selecione a opção desejada (ex: _Relatório Documentos com Arquivos_).

<figure><img src="../.gitbook/assets/image (271).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Preencha os campos Data Inicial e Data Final **conforme o intervalo de interesse**.

<figure><img src="../.gitbook/assets/image (272).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
Clique no botão verde ![](<../.gitbook/assets/image (273).png>) .
{% endstep %}
{% endstepper %}

{% hint style="success" %}
O sistema gerará automaticamente o relatório e fará o download em formato **`.xlsx`** direto para seu computador.
{% endhint %}
